Project Name: My Travel Journal 🌍✈️
It’s a digital travel journal where you can organize all your trip details in one place. You can write about locations, languages, time zones, currency, population, fun facts, and even create  a packing list.
I made it because I love to travel a lot, and I wanted a way to keep all my travel plans organized in one place.
I used HTML to structure the pages and text fields for different travel details. CSS for styling, specifically fonts. 
I struggled at first with organizing the sections in a user-friendly way and making the text boxes big enough for comfortable typing.
